# 编辑器需求与验收标准

## 1. 文字编辑

- 单击文字元素可选中。
- 双击文字元素可直接编辑内容。
- 右侧属性面板可修改：
  - 元素角色：标题、正文、图注、标签、页眉；
  - 字体；
  - 字号；
  - 字重；
  - 颜色；
  - 对齐方式；
  - 文本内容。
- 文字元素可自由移动、缩放，不影响其他元素。

## 2. 图片编辑

- 支持插入本地图片并转为 data URL。
- 图片元素支持展示样式：contain、cover、frame、circle、raw。
- 图片容器支持裁剪显示，支持对象位置和缩放控制。
- 图片可自由移动、缩放，不影响其他元素。

## 3. 图形编辑

- 支持矩形、圆角卡片、圆形、六边形、线条等基础图形。
- 图形支持填充色、描边色、描边宽度、圆角等属性。
- 图形可自由移动、缩放。

## 4. 元素隔离

- 所有可编辑元素使用绝对定位。
- 修改一个元素的位置、尺寸、层级、内容，不应导致其他元素 reflow。

## 5. 多选与批量处理

- 支持 Shift / Cmd / Ctrl 点击进行多选。
- 多选后可整体拖动。
- 多选后可整体缩放。
- 多选后可批量调整层级。

## 6. 层级处理

- 支持置于顶层。
- 支持置于底层。
- 支持上移一层。
- 支持下移一层。
- 多选时层级操作应应用到所有选中元素。

## 7. 模板与扩展性

- 内置常见版式模板：章节页、双图页、四图并列、流程页、鱼骨图、矩阵页。
- 版式模板应通过数据结构生成，而不是写死在 DOM 中，便于继续扩展。

## 8. 导入导出

- 支持导出 JSON，保留编辑状态。
- 支持导入 JSON 恢复编辑。
- 支持导出自包含 HTML，用于汇报、上传或归档。
