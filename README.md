# ppt-like-html

一个面向 HTML 汇报页的在线可视化编辑器原型，基于 VIVOID 正式汇报模板风格设计。

## 当前 MVP 能力

- 文字元素：选中后可改角色（标题/正文/图注/标签）、字体、字号、颜色、对齐、内容，并可自由拖动/缩放。
- 图片元素：可插入本地图片，支持 contain / cover / frame / circle / raw 等展示样式，可裁剪显示、缩放、移动。
- 图形元素：支持矩形、圆形、圆角卡片、线条、六边形，可自由拖动/缩放。
- 多元素：支持 Shift / Cmd / Ctrl 多选，可一起移动、缩放。
- 层级：支持置顶、置底、上移一层、下移一层，多元素可一起处理。
- 版式模板：内置章节页、双图页、四图并列、流程图、鱼骨图、矩阵页等基础版式，可继续扩展。
- 导出：支持导出自包含 HTML 和 JSON；支持导入 JSON 继续编辑。

## 本地使用

直接打开 `index.html` 即可使用。无需构建工具。

```bash
open index.html
```

## 目录结构

```text
index.html              # 编辑器入口
src/styles.css          # VIVOID 同风格界面与编辑器样式
src/app.js              # 编辑器核心逻辑
src/brand.js            # 公司 Logo data URI
/docs/EDITOR_REQUIREMENTS.md
```

## 设计原则

1. 输出内容仍然是 HTML 16:9 slide，而不是 Canvas 截图。
2. 所有元素绝对定位，单个元素修改不影响其他元素布局。
3. 先实现可用 MVP，再将编辑能力插件化。
4. 保持原汇报模板风格：米白纸面、衬线标题、墨绿/琥珀强调色、细线页眉、克制留白。
